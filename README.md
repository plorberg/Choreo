# Choreo Editor - Milestone 1

## Cloud-only / Codespaces

This milestone provides a 2D stage canvas with 16 draggable dancers and a 3D preview toggle.

### Run in GitHub Codespaces (or any cloud dev env)

1. Open the repo in GitHub Codespaces (or any cloud IDE with Node.js).
2. If not already run, open terminal and run:
   ```bash
   npm install
   npm run dev
